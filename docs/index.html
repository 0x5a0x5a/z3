UI Components

Z3 CSS provides you with classes to use which allow you to quickly construct the UI with ease everything in one place. For now it supports limited number of components such as input fields,menu,cards and so on.

Getting Started

Download the framework and embed the css file in your pages! It's that easy.

<link rel="stylesheet" type="text/css" href="path_to_z3.min.css_file">

Radios & Checkboxes

Radios

Make sure that input id and the label for attr match accordingly and must be unique. 
<input type="radio" id="r1" name="radio" checked>
<label for="r1">I'm Checked</label>

<input type="radio" id="r2" name="radio">
<label for="r2">Check Me</label>

Checkboxes

Make sure that input id and the label for attr match accordingly and must be unique. 
<div class="z3checkbox">
    <input id="c1" type="checkbox" value="1" name="check" checked>
    <label for="c1"></label>
</div>
<div class="z3checkbox">
    <input id="c2" type="checkbox" value="2" name="check">
    <label for="c2"></label>
</div>

Modals

Sleek popups to confirm an action

Are You Sure?

Dear User,

There's nothing much to be worried about if your closing this Modal Window.

JavaScript is not included in the snippet. You will need to toggle the class by yourself. 
<!-- Class to Toggle would be z3modal-active -->

    <div class="z3modal-background"></div>
    <div class="z3modal-inner">

        <div class="z3modal-body">

    </div>
</div>
